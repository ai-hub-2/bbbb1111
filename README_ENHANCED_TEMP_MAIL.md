## 🚀 كيفية الاستخدام - How to Use

### 1. تشغيل الأداة - Running the Tool
```bash
python3 real_tools/real_temp_mail.py
```

### 2. إنشاء بريد مؤقت - Creating Temp Email
- اختر الخدمة (Service)
- اختر البلد (Country)
- احصل على البريد مع كلمة المرور
- افتح الواجهة الإلكترونية تلقائياً
